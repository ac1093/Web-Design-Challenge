Cloudiness


The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities. After assemling the dataset we used matplotlib to plot various aspects of the weather vs. Latitude
